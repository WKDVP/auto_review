

### 강의평가 자동 완성 크롬 확장 프로그램입니다

+ ### 사용법:
1. 다운받은 압축 파일을 압축 해재
2. 구글 크롬 -> 상단에 점 3개 -> 확장 프로그램 -> 확장 프로그램 관리
3. 개발자 모드 켜기
4. 압축해제된 프로그램을 로드합니다 클릭
5. 압축 해제한 폴더 선택
6. 웹정보 강의평가 들어가면 자동으로 입력되고 팝업 창만 클릭하십쇼

참고로 아이콘 클릭하면 강의평 선택창이 뜹니다만
실제 적용되지는 않습니다 마치 우리 강의평처럼
